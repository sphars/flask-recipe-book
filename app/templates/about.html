About Flask Recipe Book

Flask Recipe Book is a web application for viewing and storing other user's recipes. It provides basic functions for viewing a recipe, viewing authors and their recipes, and logging in and registering for an account.

This application is written in Python using the Flask framework. It was created for a Dynamic Web Languages course.
